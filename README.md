# BHL_2024_cybersecurity

## Instalation and usage:
### Requirements:
- enabled powershell script execution
- python interpreter 3.12
- dotnet 8.0

### Building:
Simply run powershell script:
```powershell
    ./build.ps1
```

### Executin:
Simply run powershell script:
```powershell
    ./run.ps1
```
